                  # Professional Client Management System (PCMS)

A multi-tenant platform designed to manage **Doctors, Advocates, and Accountants** through a single application.

## 🚀 Features
* Multi-Tenant Architecture
* JWT Authentication & Authorization
* Role-Based Access Control
* Super Admin & Admin Management
* Dynamic Forms & Records
* Audit Logging
* Secure API Development
* Clean Architecture Implementation

## 👥 User Roles

### Super Admin

* Create Tenants
* Create Admins
* Assign Modules
* Manage System Configuration

### Admin

* Manage Users
* Manage Records
* Manage Forms
* Access Assigned Modules

## 📦 Modules

* 🏥 Doctor Management
* ⚖️ Advocate Management
* 📊 Accountant Management
